About the Project

This sign language translator application is designed to help deaf and blind individuals communicate more effectively. The system uses computer vision and machine learning to recognize sign language gestures and translate them into text.

The project is being submitted to the Prezident Tech Award in Uzbekistan as an innovative solution to improve accessibility and inclusion.

How It Works

  1. Upload sign language videos with corresponding words
  2. Process the videos to extract hand and body landmarks
  3. Train a machine learning model on the processed data
  4. Use the trained model to translate sign language in real-time

The system uses MediaPipe for hand and pose detection, and a Random Forest classifier for gesture recognition.
